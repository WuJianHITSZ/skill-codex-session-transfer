# License Decision

This repository is being prepared for public release, but no open-source license has been selected yet.

Before publishing to a public GitHub repository, replace this file with a real `LICENSE` file or remove it after choosing a license.

Common choices:

- MIT
- Apache-2.0
- BSD-3-Clause

If you do not add a license, the repository will be publicly visible but others will not automatically have permission to reuse or modify the code.
